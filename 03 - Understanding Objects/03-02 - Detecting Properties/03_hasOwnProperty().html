02 hasOwnProperty()

truthy - values that evaluate to true.

  • objects
  • a nonempty string
  • a nonzero number
  • true

falsy - values that evaluate to false.

  • null
  • undefined
  • 0
  • NaN
  • false

Using the in operator

The in operator is a more reliable way to test for the existence of a property.

The in operator checks for both own properties and prototype properties.

The in operator does NOT evaluate the value of the property.

hasOwnProperty() - is present on ALL objects, BUT returns true only if the given property exists and is an own property.
