Preventing Extensions

Object attribute[[Extensible]] - Boolean value indicating if the object itself can be modified.

ALL objects are [[Extensible]] by default.

You can check the value of [[Extensible]] by using Object.isExtensible(object);.

Create an nonextensible object withObject.preventExtensions(object);.

NOTE: Attempting to add a property to a nonextensible object will throw an error in strict mode. In nonstrict mode, the operation fails silently. You should always use strict mode with nonextensible objects so that you are aware when a nonextensible objects is being used incorrectly.
