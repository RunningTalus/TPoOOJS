Property Attributes

02 Data Property Attributes

Data Property Attributes possess QTY #2 additional values that Accessor Properties do NOT.

  1. [[Value]] - holds the property value, attribute is filled in automatically when you create a property on an object. ALL property values are stroed in [[Value]], even if the value is a function
  2. [[Writable]] - Boolean value indicating whether the property can be written to. Defaults to true.

Use Object.defineProperty(); to fully define a data property of an object.
