Understanding Objects

Enumeration

All properties that you add to an object are enumerable, AND can be iterated over with a for-inloop.

Enumerable properties have an internal [[Enumerable]] attribute set to true.

Each time through the for-in loop, the property variable is filled with the next enumerable property on the object until all properties have been used.

Best Use Case of the for-in loop: When you need to operate on property names, and you don't need an array.

The for-in loop enumerates over both own instance properties and prototype properties.
