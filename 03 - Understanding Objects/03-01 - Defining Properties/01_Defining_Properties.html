Understanding Objects

Defining Properties

[[put]] - An internal JavaScript Method used on objects when properties are intially added. The [[put]] Method creates a spot in the object to stroe the property.

own property - Indicates that the specific instance of the object owns that property.

The property is stored directly on the instance, and ALL operations on the property must be performed through that object.

[[set]] - An internal JavaScript Method used on objects when a new value is assigned to an existing property
