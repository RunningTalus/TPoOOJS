CONSTRUCTORS & PROTOTYPES

The [[Prototype]] Property

You can read the value of the [[Prototype]] property by using the Object.getPrototypeOf() method on an object.

NOTE: For any generic object like this, [[Prototype]] is always a reference to Object.prototype.

//

NOTE: __proto__ Is a property on all objects that allows you to both read from and write to the [[Prototype]] property. This on the list for inclusion in ECMAScript6

Use isPrototypeOf() method on objects to test if one object is a prototype for another.

//

You CANNOT assign a value to a prototype property from an instance.
