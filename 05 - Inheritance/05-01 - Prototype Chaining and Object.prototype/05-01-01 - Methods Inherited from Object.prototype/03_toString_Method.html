toString() Method

toString() method is called as a fallback whenever a valueOf() returns a reference value instead of a primitive value.

Example: When a string is used for one operand for the plus operator, the other operand is automatically converted to a string. If the other operand is a primitive value, it is converted to a string. If it is a reference value, valueOf() is called, if valueOf() returns a reference value- toString() is called.
