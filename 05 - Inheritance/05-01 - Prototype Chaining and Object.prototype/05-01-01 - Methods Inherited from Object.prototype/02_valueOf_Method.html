Prototype Chaining and Object.prototype

valueOf() Method

valueOf() Method is called when an operator is used on an object.

By default valueOf() returns the object instance.

Primitive wrapper types override valueOf()

  • valueOf() returns a string for string.
  • valueOf() returns a Boolean for Boolean.
  • valueOf() returns a number for Number.

The Date objects's valueOf() method returns the epoch time in milliseconds. This is the same thing that Date.prototype.getTime() does.
