Modifying Object.prototype

03 Adding properties to Object.prototype

Adding properties to Object.prototype has side effects, the add() property is enumerable, and can be iterated over with a for-in loop.

Crockford recommends using hasOwnProperty(), for for-in loops all the time. This limits the use of for-in loops to own properties.

LESSON LEARNED: DO NOT MODIFY Object.prototype.
