PRIMITIVE & REFERENCE TYPES

Identifying Reference Types

typeof

typeof operator use on functions will return or log "function".

typeof on all other reference types will return or log "object"

instanceof

instanceof operator takes QTY#2 parameters

  1. Object
  2. Constructor

When the value is an instance of the type that the constructor specifies, instanceof will return true, otherwise it returns false.

The instanceof operator can identify inherited types.

WHY?

Every object is an instance of Object because EVERY reference type inherits from Object.

Each reference type is correctly identified as an instance of Object, from which all reference types inherit.

NOTE: object instanceof Array logs FALSE.

Arrays are objects, but objects are NOT Arrays
