PRIMITIVE & REFERENCE TYPES

Identifying Arrays

instanceof operator can identify arrays.

This becomes problematic in the browser, as each webpage has its own global context- and thus its own version of built-in types: (Object, Array...)

Array.isArray();

In ECMAScript 5 compliant environments use Array.isArray();

Array.isArray(); will return true when it receives a value that is a native array from ANY context.

Compatibility

Supported in node.js

NOT supported in IE8 and earlier.
